Installation
============

Using PyPI
----------

.. code-block:: bash

    pip install liso


From Source
-----------

.. code-block:: bash

    git clone https://github.com/zhujun98/liso.git
    cd liso
    pip install -e .


Install optional 3rd party optimization libraries
-------------------------------------------------

pyOpt
~~~~~

.. code-block:: bash

    $ git clone https://github.com/zhujun98/pyOpt
    $ cd pyOpt
    $ sudo pip3 install .
